# üåç Global Variables for Advanced Snake Game

The `config.toml` file is used to configure the game settings. Here‚Äôs what each variable does:

---

- `display.screen_size.x` & `display.screen_size.y` : Defines the screen width and height in pixels.
- `display.grid_size.x` & `display.grid_size.y` : Defines the number of grid cells in the X and Y direction.

- `game.tps` : Controls the game speed (higher value = faster game)

- `apples.number` : Number of apples generated at the start.
- `apples.power` : Defines the power value of an apple.

- `textures.apples` : Specifies the file path for the food (apple) texture.
- `textures.walls` : Specifies the file path for the wall textures.
- `textures.background` :  Specifies the file path for the background textures.

- `walls.map` : Path to the CSV file that defines the wall placement in the game.

- `keys.pause` : Key to pause the game
- `keys.exit` : Key to exit the game
